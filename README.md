## 🔥 Project Overview

**DAAM Portal** is a modern and user-friendly support platform built to offer a smooth and interactive experience.
Created based on professional guidance, suggestions, and detailed feedback to ensure clarity and usability.

✨ **Key Features:**

* Responsive design for all devices 📱💻
* Arabic & English language support 🌍
* Dark/Light theme toggle 🌙☀️
* Easy-to-use interactive main menu 🗂️
* Clean, modern UI for seamless navigation 🎨

## 🛠️ Installation

To run the project locally:

1. Clone the repository

```bash
git clone https://github.com/raya19998333/daam.git
```

2. Go to the project folder

```bash
cd daam
```

3. Open `index.html` in any browser!
